# kotlin-cli-maven-spring-surefire-jacoco-cucumber-junit-hello-world

## Description
A POC for spring app using junit5, cucumber,
and jacoco framework with surefire plugin.

This is a simple and trivial way to start:
  - kotlin
    - springframework
    - cucumber test runner for junit5
  - cucumber
    - parameterized scenario

## Tech stack
- kotlin
- maven
  - spring
  - junit5
  - cucumber
  - surefire
  - jacoco

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
